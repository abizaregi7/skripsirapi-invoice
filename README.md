# SkripsiRapi Invoice Generator

## Cara Menjalankan

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Jalankan aplikasi:
```bash
streamlit run app.py
```

3. Buka browser di `http://localhost:8501`

## Catatan
- File `logo.jpg` harus berada di folder yang sama dengan `app.py`
- Pastikan Python 3.9+ sudah terinstall
